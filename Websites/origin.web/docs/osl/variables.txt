variablename = "assign-data"
sets a variable to the inputted data

list.[itemnumber] = "assign-data"
Sets the selected item of a list to the inputted data

object.key("key") = "assign-data"
Sets the selected key of a json object to the inputted data

---

You can also use modifiers for assignment such as:

variable += 10
This code will either append ten on the end of a string or add ten to a number data type

---

This code only works on number data types and will error or any non number type

variable *= 5
Sets variable to itself multiplied by 5

variable /= 5
Sets variable to itself divided by 5

variable -= 5
Sets variable to itself minus 5
