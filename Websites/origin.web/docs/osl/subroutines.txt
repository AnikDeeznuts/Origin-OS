def “name” “input1,input2,input3,input4” 
opens a define command

this command allows you to create commands 
that you can reuse in your main script

-------------------------------------------

endef
closes the current define command

-------------------------------------------

Event "condition"
opens an event command

when the condition is met, anything inside 
of the current event command will be run

-------------------------------------------

Endevent
closes the current event
